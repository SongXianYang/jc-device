http://localhost:9090/swagger-ui.html
###数据库表

###项目目录说明
config   配置文件  
controller    响应的用户请求的url  
dto   数据传输  
entity   实体类与数据库表字段映射层  
http    集成httpclient需要用到的java文件  
mapper   数据持久层  
service   业务层  
utils   工具类  
JcDeviceApplication   主程序入口  
resources   静态资源文件     
  mapper    xml映射文件   
  static    静态资源   
  application.yml  配置   
test    测试
